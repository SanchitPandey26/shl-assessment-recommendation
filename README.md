# SHL Assessment Recommendation System

This repository contains:

- `backend/` → FastAPI-based recommendation engine  
- `frontend/` → Next.js web interface  
- `notebooks/` → evaluation and analysis  
- `docs/` → approach documentation for SHL

Tech stack:
- Python + FastAPI + SentenceTransformers + Chroma
- Next.js + Vercel
- Render deployment for backend

Follow the `/backend` and `/frontend` READMEs for setup.
